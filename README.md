AstroVision: Astrophysical Image Classification using CNN

Overview
This project applies deep learning to classify astrophysical images into categories such as asteroid, black hole, and galaxy.

 Models Used
- Custom Convolutional Neural Network (CNN)
- VGG16 Transfer Learning Model

 Techniques
- Image preprocessing
- Data augmentation
- Normalization
- Transfer learning
- Model evaluation and comparison

 Results
- VGG16 achieved strong performance with approximately 92–96% F1-score
- The transfer learning approach outperformed the baseline CNN model

Tools
Python, TensorFlow, NumPy, matplotlib, Google Colab
